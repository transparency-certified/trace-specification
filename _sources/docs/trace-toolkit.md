# TRACE Toolkit

The TRACE toolkit is a Python library/package that implements the TRACE
specification. 

## `trace-cli`

`trace-cli` supports:
* Submit
* Inspect
* Verify

## `trace-bdbag`

`trace-bdbag` package:
* Create and sign TRO

## `trace-server`

`trace-server`:
* Expansion of the proof-of-concept for containerized execution not based on WT.
